# Colab

This directory contains Google Colab-based notebooks, training experiments, setup notes,
and supporting documents used throughout the `mlgym` repository.

## Purpose

This section is for Colab-first workflows, including:

- training notebooks
- quick experiments
- environment setup notes
- GPU/runtime-specific work
- temporary or exploratory prototypes that may later move elsewhere in the repo

## Contents

This directory may include:

- Colab notebooks from google's training docs
- Colab notebooks for model training
- one-off experiments and prototypes
- notes on runtime behavior and limitations
- setup instructions for Drive, GitHub, and dependencies
- reusable notebook templates
- small benchmark or debugging notebooks

## Goals

- keep Colab-based work organized in one place
- separate exploratory notebook work from more stable project code
- make experiments easy to resume, compare, and version
- document Colab-specific setup and workflow details

## Notes

- Colab notebooks should be saved frequently and synced to GitHub when useful.
- Large datasets and generated artifacts should generally not be committed unless intentional.
- Use this directory for notebook-centric work; reusable library code can live elsewhere in the repository.
