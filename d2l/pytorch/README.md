# PyTorch

This directory contains my PyTorch-based work for [Dive into Deep Learning](https://d2l.ai/).

## Purpose

This section is for working through D2L using the PyTorch backend. It includes notebooks,
exercises, notes, and small experiments organized by chapter.

## Layout

```text
pytorch/
├── README.md
├── ch03/
├── ch04/
├── ch05/
├── notes/
└── ...
```

## Directory convention

A typical chapter directory may contain:

```text
pytorch/ch03/
├── 3_1_linear_regression.ipynb
├── 3_2_linear_regression_scratch.ipynb
├── exercises.md
└── notes.md
```

## Contents

This directory may include:

- D2L chapter notebooks adapted for PyTorch
- exercise solutions
- implementation notes
- debugging notes
- small side experiments
- summaries of key concepts

## Goals

- learn deep learning concepts through PyTorch
- understand both high-level APIs and lower-level tensor operations
- keep chapter work organized and easy to revisit
- track progress in a clean, version-controlled structure

## Suggested workflow

1. Open the relevant D2L chapter
2. Run the PyTorch notebook in Colab or locally
3. Save the notebook under the appropriate chapter directory
4. Add notes and exercise answers
5. Commit progress regularly

## Naming suggestions

Use consistent names where possible:

- notebooks: `3_1_linear_regression.ipynb`
- notes: `notes.md`
- exercises: `exercises.md`

This makes it easier to compare progress across backends later.
