# Dive into Deep Learning (D2L)

This contains study material, notebooks, notes, and exercises for working through
[Dive into Deep Learning](https://d2l.ai/).

## Layout

This directory is organized by deep learning backend.

```
d2l/
├── README.md
├── pytorch/
├── tensorflow/
├── jax/
└── ...
```

## Directory convention

Each backend directory should mirror a similar structure where possible, for example:

```
d2l/pytorch/
├── ch03/
└── notes/
```
```
mlgym/d2l/tensorflow/
├── ch03/
└── notes/
```
```
mlgym/d2l/jax/
├── ch03/
└── notes/
```

## Contents

Each backend folder may include:

- notebooks based on D2L chapters
- exercise solutions
- code experiments
- implementation notes
- backend-specific observations

## Goals

- study D2L across multiple frameworks
- compare API design and implementation differences
- keep progress organized by backend
- maintain a clean, version-controlled learning record

## Suggested workflow

1. Open the relevant D2L notebook
2. Run and modify it in Colab or locally
3. Save it under the appropriate backend directory
4. Add notes and exercise answers
5. Commit changes regularly
